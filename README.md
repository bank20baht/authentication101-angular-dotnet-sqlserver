# Authentication 101

Simple Authentication by Angular + .NET Fullstack Project Templete

## installation

```
git clone git@github.com:bank20baht/authentication101-angular-dotnet-sqlserver.git
cd authentication101-angular-dotnet-sqlserver
docker-compose up
```

and using init.sql at root to init database.

## Tech Stack
- Angular 21 standalone
- .NET 10 minimal API + MediatR
- MS SQL Server 2022
- Docker
- i18n for en-th website
- daysiui
- tailwindcss

## api docs

if you run docker-compose go to http://localhost:4000/swagger/index.html
